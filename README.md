## 它到底怎么工作的

说起来简单到令人怀疑：

把鼠标移到哪个屏幕，那个屏幕就亮。移开了，就变暗。

仅此而已。没有快捷键，没有手动切换，就是跟着光标走。你的注意力在哪儿，亮光就在哪儿，其他地方变成一片"舞台布景"，自然就退到了背景里。

## 可以调哪些参数

ScreenFocus 不是那种"装上就不管了"的工具，它给了几个关键的自定义选项：

- **调暗程度**：从微微变暗到接近全黑，自己拖滑块
- **调暗延迟**：光标离开多久之后才开始变暗，避免快速切换时的频繁闪烁
- **渐变速度**：变亮和变暗各自的过渡时长，流畅不突兀
- **按显示器开关**：某块屏幕不想被调暗？单独关掉它
- **主屏设置**：默认主屏不参与调暗，可以手动改

设置完基本就不用再动了。

## 两个你可能没想到的好处

**一、保护眼睛**

多屏同时全亮，等于你的眼睛一直在承受好几块屏幕加起来的光线。时间长了眼睛酸、干、累是正常的。ScreenFocus 把暂时不用的屏幕调暗，实际上是在减少总光照量，对长时间盯屏的人来说挺有实际意义。

**二、延长显示器寿命**

屏幕常亮时间越长，背光寿命消耗越快。暂时用不到的副屏一直亮着，纯粹是浪费。调暗不等于关闭，不影响使用，但能明显减少不必要的损耗。
